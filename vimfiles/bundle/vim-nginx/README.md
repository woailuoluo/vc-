# nginx-contrib-vim

It's a clone of https://github.com/nginx/nginx/tree/master/contrib/vim.

Then it's easier to use via [Vundle](https://github.com/VundleVim/Vundle.vim):

    Plugin 'moskytw/nginx-contrib-vim'
    
Have fun. :)
